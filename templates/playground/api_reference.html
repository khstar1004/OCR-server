API Reference

Army-OCR API Reference

이 문서는 Army-OCR을 외부 시스템에서 호출하기 위한 범용 OCR API 계약을 정리합니다. 고객 연동은 /api/v1/*가 기준이며, 체험 UI 경로는 데모와 QA 용도입니다.

구현 기준 이 페이지는 현재 서비스의 /openapi.json에 등록된 endpoint만 설명합니다. 외부 계약에 없는 내부 전용 endpoint는 Reference에 넣지 않습니다.

기본 정보

항목값
Preview Basehttp://14.50.225.33:18110
API Prefix/api/v1
OpenAPI/openapi.json
인증현재 preview는 별도 API key 없음. 운영 공개 시 reverse proxy 또는 gateway 인증 권장
지원 입력PDF, PNG, JPG, JPEG, WEBP
좌표계bbox=[x1,y1,x2,y2], 페이지 이미지 픽셀 기준
비동기 공통 상태processing, complete, failed

Runtime Settings

운영 설정 조회/저장

GET/playground/api/admin/runtime-settings
PUT/playground/api/admin/runtime-settings

관리자 세션이 필요합니다. 이미지를 다시 만들지 않고 OCR upstream, timeout, 동시성, OCR prompt/batch, playground 제한, 국회 API/LLM/감시 주기 같은 운영값을 저장합니다. restart_required인 vLLM 실행 인자는 저장 후 vLLM만 재시작하면 반영됩니다.

curl -c cookies.txt -X POST http://127.0.0.1:18110/playground/api/auth/login \
  -H "content-type: application/json" \
  -d "{\"username\":\"admin\",\"password\":\"<admin-password>\"}"

curl -b cookies.txt -X PUT http://127.0.0.1:18110/playground/api/admin/runtime-settings \
  -H "content-type: application/json" \
  -d "{\"values\":{\"ocr_service_timeout_sec\":300,\"target_api_timeout_sec\":60}}"
Field설명
path설정 파일 경로. Docker/k8s 기본값은 /data/runtime/runtime-config/settings.json
specs[]설정 키, 타입, 최소/최대값, 재시작 필요 여부
values현재 effective value
overridesUI/API로 저장한 override 값

Synchronous OCR

이미지 1장 OCR

POST/api/v1/ocr/image

이미지 파일 하나를 즉시 OCR하고 페이지 텍스트, Markdown, 블록, bbox를 반환합니다.

Form field기본값설명

curl -X POST http://127.0.0.1:18110/api/v1/ocr/image \
  -F "file=@page.png" \
  -F "page_number=1" \
  -F "include_raw=false"

결과 조회 응답 주요 필드

Field설명

import requests

base_url = "http://127.0.0.1:18110"

with open("sample.pdf", "rb") as file:
    response = requests.post(
        f"{base_url}/api/v1/marker",
        files={"file": file},
        data={
            "output_format": "json,markdown,html,chunks",
            "mode": "balanced",
            "page_range": "0",
        },
    )

request_id = response.json()["request_id"]
result = requests.get(f"{base_url}/api/v1/marker/{request_id}")
print(result.json())
File to Document
import requests

base_url = "http://127.0.0.1:18110"

with open("sample.pdf", "rb") as file:
    file_res = requests.post(
        f"{base_url}/api/v1/files",
        files={"file": file},
    ).json()

doc_req = requests.post(
    f"{base_url}/api/v1/create_document",
    json={"file_id": file_res["file_id"]},
).json()

doc_result = requests.get(doc_req["request_check_url"]).json()
document_id = doc_result["document"]["document_id"]
