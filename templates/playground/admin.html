운영 메모

Playground는 국회 OCR demo와 분리된 범용 OCR 체험/관리 화면입니다.

  • 운영 설정 저장은 Docker 이미지를 다시 만들지 않습니다.
  • 일반 설정은 새 요청부터 반영됩니다.
  • vLLM 실행 인자는 저장 후 vLLM 컨테이너 또는 Pod만 재시작합니다.

계정 승인

사용자 신청을 검토하고 승인 또는 반려합니다.

운영 설정

OCR API, Chandra, playground, vLLM 값을 runtime JSON으로 저장합니다.
